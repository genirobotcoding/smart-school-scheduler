# 우리학교 스마트 스케줄러

중등 수업용으로 만든 학교 일정 관리 웹앱입니다.

## 실행 방법

`index.html` 파일을 브라우저로 열면 바로 실행됩니다.

## 기능

- 학교 검색
- 오늘 급식 카드
- 이번 주 학사일정 카드
- 시험, 고사, 평가 일정 기반 D-Day 표시
- 날짜가 있는 개인 할 일 추가, 완료 체크, 삭제
- 저장된 학교를 기본 대시보드로 사용하고 설정에서 학교 변경
- API 키가 없을 때 데모 데이터로 시연

## 실제 학교 데이터 사용

나이스 교육정보 개방 포털에서 Open API 키를 발급받아 앱 상단의 `나이스 API 키` 입력칸에 넣으면 실제 학교 검색, 급식, 학사일정 조회를 시도합니다.

사용 API:

- `schoolInfo`
- `mealServiceDietInfo`
- `SchoolSchedule`
